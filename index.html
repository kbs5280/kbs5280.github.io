Projects

My flagship project for this year is PreQ™, a LendingTree-style web application. It allows a user to enter a single set of personal information and receive loan pre-qualification offers from four industry-specific lenders.

Under-the-hood PreQ™ validates, sanitizes and formats data for four unique APIs, including three REST APIs and a SOAP API. Data is formatted as JSON, XML and URLEncoded strings customized for each API. Requests are posted via web-standard HTTPS and utilize OAUTH, MTLS, and AES256 bit encryption to securely transmit data.

The app is built in Salesforce and features a password-protected user interface built on Salesforce Sites. The controllers and API services are built in Apex and utilize SOQL to perform DML operations on an extensive relational database.

*PreQ™ was built while I was an employee of - and is the exclusive intellectual property of - ClearChoice Management Services.
